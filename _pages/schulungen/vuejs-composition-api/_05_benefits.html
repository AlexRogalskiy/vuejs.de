Die Composition API effektiv einsetzen

In unserer Vue.js Schulung erfahrt Ihr, warum wir die neue API brauchen und wie Ihr sie am besten in Euren Projekten einsetzt. Natürlich wird auch in diesem Kurs der Praxisanteil über 60 % der Zeit betragen. Schließlich wollen wir nicht nur über die Composition API reden, sondern sie vor allem verwenden. Dazu gibt es viele praktische Aufgaben aus der Praxis. Auf diese Weise erkennt Ihr, warum die Composition API eine tolle Möglichkeit ist, Anwendungslogik zwischen Komponenten in Vue.js zu teilen.
